About The Project

Frameworks

For this project, we utilized NodeJS and multiple packages. The only javascript framework utilized was JQuery for simple data manipulation. We used the express package to create various routes for the server. We used multiparty to parse form data received from the client. The package serve-favicon is used by express to load the favicon correctly. To handle PUT requests, we used body-parser to correctly parse the body of the request and update the database.
